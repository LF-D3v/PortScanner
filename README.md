## Installation

Clone the repository into a folder
```
git clone https://github.com/LF-D3v/PortScanner.git
```

Move to the project folder
```
cd PortScanner
```

Install the requirements
```
pip3 install -r requirements.txt
```

Execute the program
```
python3 portscanner.py
```
